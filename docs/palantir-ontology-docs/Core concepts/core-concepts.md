# Core concepts
有几个概念是理解 Foundry Rules 的基础：

There are several concepts that are fundamental to Foundry Rules:
* **[Object model](/docs/foundry/foundry-rules/object-model/):** Foundry rules 是具有 **properties** 的 **objects**，可以通过 **rule proposals** 进行管理。

* **[Workshop application](/docs/foundry/foundry-rules/workshop-application/):** Foundry rules 由一个 **Workshop application** 进行管理，该 application 通常包含 **Rule Editor** 和 **Proposal Reviewer** 功能。

* **[Rule logic](/docs/foundry/foundry-rules/rule-logic/):** Foundry rules 编码了旨在执行[特定工作流](/docs/foundry/foundry-rules/overview/#example-use-cases) 的业务逻辑。该逻辑遵循特定的结构，并定义了一组对数据的 transformations。

* **[Foundry Rules workflow configuration](/docs/foundry/foundry-rules/foundry-rules-workflow-configuration/):** Foundry rules 使用 workflow configuration editor 进行配置。该编辑器是 workflow owners 用于根据其用例定制 Foundry Rules 的 interface。

![Foundry Rules workflow overview list](/docs/resources/foundry/foundry-rules/overview@2x.png)

* **[Object model](/docs/foundry/foundry-rules/object-model/):** Foundry rules are **objects** with **properties** that can be managed by **rule proposals**.
* **[Workshop application](/docs/foundry/foundry-rules/workshop-application/):** Foundry rules are managed by a **Workshop application** which usually includes **Rule Editor** and **Proposal Reviewer** functionality.
* **[Rule logic](/docs/foundry/foundry-rules/rule-logic/):** Foundry rules encode business logic designed to perform a
[particular workflow](/docs/foundry/foundry-rules/overview/#example-use-cases). This logic follows a specific structure and defines a set of transformations upon data.
* **[Foundry Rules workflow configuration](/docs/foundry/foundry-rules/foundry-rules-workflow-configuration/):** Foundry rules are configured using the workflow configuration editor. The editor is an interface used by workflow owners to customize Foundry Rules to their use case.

![Foundry Rules workflow overview list](/docs/resources/foundry/foundry-rules/overview@2x.png)
