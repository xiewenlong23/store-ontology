展示 FAD 命令主界面：

- `/fad:pipeline` - 默认端到端流程
- `/fad:help` - 命令概览
- `/fad:map-codebase` - 项目架构/约定映射
- `/fad:optimize` - review 后优化
- `/fad:quality-gate` - 严格发布门禁
- `/fad:pr-branch` - 准备干净的 PR 分支
- `/fad:ship` - 最终发布流程
- `/review` - 代码审查

推荐起始命令：`/fad:map-codebase`（首次）或 `/fad:pipeline`（新任务）

审计日志目录：`.planning/audit/`
