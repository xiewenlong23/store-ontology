# PRD.md - Product Requirements

## 项目: 门店大脑 AI 原生应用

### 1. 概述

飞书小程序为前端的 AI 原生应用，通过 TTL 本体定义业务语义层，AI 通过 SPARQL 查询本体 + 推理执行任务。

### 2. 核心场景

#### 当前：临期打折
- 商品健康度监控
- 自动折扣推荐（5 档 tier）
- 任务创建和跟踪
- 售罄率统计

#### 未来扩展
- 智能补货
- 盘点自动化
- 排班优化

### 3. 功能需求

- [ ] 商品列表展示（按类别分组）
- [ ] 临期商品标记（红/黄/绿）
- [ ] AI 折扣推荐
- [ ] 折扣任务创建/管理
- [ ] 意图识别对话（查/创/报）
- [ ] 本体知识库（TTL）

### 4. 非功能需求

- 响应时间 < 200ms
- 本体推理准确性 > 90%
- 代码覆盖率 > 70%
