# Create and save maps
## Create a new map
要创建地图，请展开左侧的 Foundry 导航栏，然后在 Apps（应用程序）部分中单击 **View all（查看全部）**。您将在 **Operational Applications（运营应用程序）** 部分下找到 **Map（地图）** 应用程序。

To create a map, expand the left-hand side Foundry navigation bar, then click on **View all** in the Apps section. You will find the **Map** application under the **Operational Applications** section.

## Save a map
要首次保存地图，请单击屏幕左上角的 **Save as...（另存为...）**，然后选择保存地图的位置。

To save a map for the first time, click **Save as...** in the top-left corner of the screen and then select a location to save the map.
![Map Save As button](/docs/resources/foundry/map/map-overview-save-as-button.png)
要在之后保存地图，请单击屏幕左上角的 **Save（保存）**。地图不会自动保存。

To save a map subsequently, click **Save** in the top-left corner of the screen.  Maps do not save automatically.
![Map Save button](/docs/resources/foundry/map/map-overview-save-button.png)
