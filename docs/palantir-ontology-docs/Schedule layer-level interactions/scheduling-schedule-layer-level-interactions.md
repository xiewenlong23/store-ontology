# Schedule layer-level interactions
Schedule layers 不仅可以在视觉上配置，还可以拥有自己定制的一组 interactions。由于同一个 object type 可以在多个 schedule layers 中使用，这意味着您可以创建高度具体的 interaction 定义。您可以使用 schedule layer 级别的 interactions，使用户能够修改 schedule objects 的某些 properties、创建新的 schedule objects，或在您的 Workshop 模块中触发 events。您还可以自定义 drag-and-drop 行为和粒度。这些选项在每个 schedule layer 配置下的 **Drag & Drop** 和 **Interactions** 子部分中进行组织，位于 **Input Data (Pucks)** 之下。

Schedule layers are not only visually configurable, but can have their own customized set of interactions. As the same object type can be used across multiple schedule layers, this means you can create highly specific interaction definitions. You can use schedule layer-level interactions to enable users to modify certain properties of your schedule objects, create new schedule objects, or trigger events in your Workshop module. You can also customize the drag-and-drop behavior and granularity. These options are organized under the **Drag & Drop** and **Interactions** sub-sections within each schedule layer's configuration under **Input Data (Pucks)**.
